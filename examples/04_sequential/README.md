# 04 Sequential

This section demonstrates multi-stage sequential pipelines.

## Files

- `relax_then_dos.py`: Stage-chained VASP relaxation and DOS.
- `mixed_dos_sources.py`: One pipeline combining `structure_from` and explicit `structure` DOS sources.

## Run

```bash
python examples/04_sequential/relax_then_dos.py
python examples/04_sequential/mixed_dos_sources.py
```
