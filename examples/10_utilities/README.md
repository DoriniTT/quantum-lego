# 10 Utilities

This section contains utility and output-format demos.

## Files

- `demo_rich_console.py`: shows Rich terminal formatting utilities used by result printers.
- `error_handling.py`: demonstrates error-handling patterns — status polling, result validation, WorkGraph stage inspection, and automatic retry — without running real calculations.

## Run

```bash
python examples/10_utilities/demo_rich_console.py
python examples/10_utilities/error_handling.py
```
