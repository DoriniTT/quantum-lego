# 01 Getting Started

This section covers the smallest runnable examples.

## Files

- `single_vasp.py`: Submit one static VASP calculation with `quick_vasp`.
- `check_results.py`: Inspect status and extracted energies from a WorkGraph PK.

## Run

```bash
python examples/01_getting_started/single_vasp.py
python examples/01_getting_started/check_results.py <PK>
```
